Design of 64x32 SRAM using 90nm CMOS Technology:

In this project a 64x32 cache memory was designed in 90nm CMOS technology using Synopsys custom designer. We started from Top-level architecture describing the blocks required like SRAM array, controller, and sense amplifier, read and write buffer, Decoder. Developed schematics with suitable sizing of transistor followed by physical design and then DRC & LVS check individual and top-level and ended up having Area of 32Um2 static power 257uW Dynamic power @500Mhz 6.05mW

Fixed Block Size Motion Estimator using Verilog coding :

The objective of the project is to use Motion estimation using bandwidth compression method. It is a part of predictive coding used in latest hybrid video coding standards such as H.261 and MPEG. This circuit consists of a Controller, Memory, Processing Element, and Comparator. The code for these blocks were written and tested as a part of this project.
